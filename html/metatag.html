meta태근느 html 문서에 대한 메타 데이터를 정의한다. (데이터에 대한 데이터 ==> 정보) meta태그는 항상 head 태그의 안에 들어가며, 일반적으로 문자 세트, 페이지 설명,키워드 문서의 작성자 및 뷰포트 설정을 지정하는데 사용 meta태그를 사용하는 이유 : 웹페이지에 대한 정보를 제공하므로 검색엔진이 페이지를 검색할 때 참고할 수 있고, 검색 결과에도 반영 meta 태극 ㅏ제공하는 메타데이터의 유형 & 속성 charset : 문자 세트 http-equiv : 콘텐츠 속성 정보에 대한 http 헤더 name : 문서 정보 content : 메타 데이터 내용 + 기타 여러가지 있는데. 자주 쓰는 거에만 익숙해지면서 감만 익혀보자. (사실 자동완성 하는것에 다 포함되어있음)
